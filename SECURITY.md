# Security Policy

## Supported Versions

Currently, only the latest major release of this library receives active security updates.

## Reporting a Vulnerability

Security is a top priority for the Lite ecosystem. If you discover a vulnerability, please do not open a public issue. 

Instead, please send a direct email or private message outlining the exploit. Please include:
* A description of the vulnerability.
* Steps to reproduce.
* Potential impact.

You should receive a response within 48 hours acknowledging the report and outlining the next steps for a patch and advisory release. Thank you for helping keep the ecosystem secure.
